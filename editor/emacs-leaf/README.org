** Bootstrapping

Either =nix run= or Emacs command can be used to generate =init.el= for initial startup:

#+BEGIN_SRC sh
$ nix run .#tangle
#+END_SRC

#+BEGIN_SRC sh
$ emacs --batch --eval "(require 'org)" --eval '(org-babel-tangle-file "init.org")'
#+END_SRC
